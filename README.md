# Gym Progress Tracker

A simple Android app to track gym workouts, built with **Jetpack Compose** and **Room Database**. The app allows you to manage gym machines, create workout sessions, and track exercises with weight, reps, and sets.

## Features

- **Manage Machines**: Add, edit, and delete gym machines.
- **Create Sessions**: Add new workout sessions with custom names and dates.
- **Add Exercises**: Assign machines to sessions, including weight, reps, and sets.
- **Edit Sessions**: Update exercises or add more machines to existing sessions.
- **View Sessions**: See all sessions and their exercises in a list.
- **User-Friendly UI**: Built entirely with Jetpack Compose for a modern, responsive design.

## Screens

1. **Main Menu**: Navigate to add machines, start a new session, or view existing sessions.
2. **Add Machine**: Add or edit machines with a scrollable list of existing machines.
3. **New Session**: Create a new workout session and add exercises.
4. **Edit Session**: Edit or add exercises to an existing session.
5. **View Sessions**: Browse all sessions and their exercises with options to edit or delete.

## Technologies

- Kotlin
- Jetpack Compose
- Android Room Database
- Coroutines for asynchronous database operations
- Navigation Compose for screen navigation
